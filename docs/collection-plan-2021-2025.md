## 0. 规划说明

### 0.1 规划范围

**时间范围：** 2021-2025年（近五年）  
**会议类型：**
- GDC (Game Developers Conference)
- Unreal Fest
- Unite (Unity)
- UWA Day

### 0.2 任务设计原则

**按照新规范：**
- ✅ 任务粒度：单个任务最多5个文档，推荐1个
- ✅ 可执行性：必须包含具体信息（文档列表、URL、搜索关键词等）
- ✅ 特殊权限：需要GDC Vault等权限的任务标记为 `skipped`
- ✅ 优先级：1-5（1最高，5最低）

### 0.3 任务类型

**主要任务类型：**
1. **content_supplement** - 内容补充：补充已有文档的内容
2. **create_document** - 新建文档：创建新文档（需要提供具体列表）
3. **url** - URL处理：处理特定URL
4. **topic** - 主题搜索：搜索特定主题
5. **skipped** - 已跳过：需要特殊权限

## 6. 任务生成规则

### 6.1 内容补充任务（content_supplement）

**规则：**
- 每个任务最多5个文档
- 必须提供 `target_files` 列表
- 推荐每个文档一个任务

**示例：**
```json
{
  "type": "content_supplement",
  "target_files": [
    "docs/gdc/talks/2025/gdc-2025-keynote-state-of-unreal.md"
  ],
  "priority": 1
}
```

### 6.2 新建文档任务（create_document）

**规则：**
- 每个任务最多5个文档
- 必须提供 `document_list` 或 `schedule_url` 或 `search_keywords`
- 推荐每个文档一个任务

**示例：**
```json
{
  "type": "create_document",
  "document_list": [
    {"title": "GDC 2024: State of Unreal", "file": "docs/gdc/talks/2024/gdc-2024-keynote-state-of-unreal.md"}
  ],
  "priority": 2
}
```

### 6.3 需要特殊权限的任务（skipped）

**规则：**
- 必须标记 `requires_permission: true`
- 必须标记 `status: "skipped"`
- 必须提供 `reason` 说明

**示例：**
```json
{
  "type": "content_supplement",
  "requires_permission": true,
  "status": "skipped",
  "reason": "需要GDC Vault账号访问完整视频和PPT",
  "priority": 5
}
```
