# GDC 资料索引

> 历年游戏开发者大会（Game Developers Conference）演讲资料归档 - 以单个分享为粒度，涵盖所有类型

## 0. 归档原则

本知识库以**单个演讲为粒度**进行归档，尽量保持原始数据的完整性，建立丰富的、具备细节的知识仓库。

**归档特点：**
- 每个演讲独立文档，保留完整元数据
- 尽量保持原始内容，不过度提炼
- 详细的资源链接和参考资料
- 清晰的分类和标签系统
- **涵盖所有类型：** 技术、美术、策划、运营、制作等

## 9. 使用说明

### 9.1 添加新演讲
1. 使用 `TEMPLATE.md` 作为模板
2. 创建新文档，文件名格式：`gdc-[年份]-[演讲者]-[主题].md`
3. 按主题分类放入对应目录（如 `programming/`, `graphics/` 等）
4. 同时可以按年份放入对应年份目录（可选）
5. 更新本索引文件

### 9.2 文档维护
- 尽量保持原始数据的完整性
- 补充详细的元数据和资源链接
- 记录收集日期和状态
- 定期更新内容

### 9.3 检索方式
- 按主题查找：查看"按主题分类索引"章节
- 按年份查找：查看"按年份索引"章节
- 按演讲者查找：查看"按演讲者索引"章节
- 按标签查找：查看"按标签索引"章节
