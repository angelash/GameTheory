# Unreal Circle 资料索引

## 0. 关于Unreal Circle

Unreal Circle是虚幻引擎官方举办的全国巡回线下沙龙，旨在与开发者面对面交流，分享技术经验，提供与Epic Games工程师和本地开发团队技术人员直接交流的机会。

**核心价值：**
- 面对面技术交流
- 本地化技术分享
- 开发者社区建设
- Epic Games官方支持
