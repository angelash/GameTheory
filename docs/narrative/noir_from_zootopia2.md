# 从《疯狂动物城2》走进 Film Noir 世界

> Noir 不是一个滤镜，而是一套讲“灰度故事”的方法。

## Noir 的几个关键特征

1. 城市是活的系统，会主动施压（权力结构、帮派、警局等）
2. 角色大多处在灰度地带，而非完全善恶
3. 真相曲线是“越查越不对劲”，而不是越来越光明
4. 强对比光影和压抑氛围（可以通过场景、镜头、UI、音乐体现）

## 游戏设计中的应用

- 系统层面：
  - 城市 / 派系 / 舆论作为长线变量，影响剧情与任务
  - 证据碎片系统，真相非线性呈现
- 叙事结构：
  - 每一章节解决一个案子，同时暴露更大的阴影
  - 玩家在“自保 vs 揭露真相”之间不断做选择

> 对你来说，这篇更像是一套“做 Noir 氛围游戏”的设计骨架，随时可以拿来复用。
