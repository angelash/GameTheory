# Unreal Fest 资料索引

> Epic Games主办的虚幻引擎年度盛会资料归档

## 0. 关于Unreal Fest

Unreal Fest是由Epic Games主办的年度盛会，汇聚全球虚幻引擎开发者，分享最新技术、案例研究和最佳实践。

**核心价值：**
- 虚幻引擎最新技术分享
- 实际项目案例解析
- 开发者社区交流
- Epic Games官方技术指导

---

## 1. 历年Unreal Fest概览

### 1.1 Unreal Fest 2025

**Unreal Fest Orlando 2025**
- **时间：** 2025年6月2日-5日
- **地点：** 美国奥兰多
- **亮点：** Tim Sweeney主题演讲，Unreal Engine 5.6最新动态

**Unreal Fest Tokyo 2025**
- **时间：** 2025年11月25日
- **地点：** 日本东京
- **时长：** 2天

**Unreal Fest Shanghai 2025**
- **时间：** 2025年8月22日-23日
- **地点：** 上海宝华万豪酒店

### 1.2 Unreal Fest 2024

**全球多地举办：**
- 黄金海岸：5月29-30日
- 布拉格：6月18-20日
- 西雅图：9月30日-10月3日
- 首尔：8月28-29日
- 上海：9月5-6日
- 东京：11月2日

### 1.3 Unreal Fest Shanghai 2023

**时间：** 2023年11月30日-12月1日  
**地点：** 上海前滩香格里拉酒店  
**内容：** 多场主旨演讲与技术分享
