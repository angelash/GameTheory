# 游戏为什么不平衡 & 如何解决不平衡问题

## 不平衡的常见表现

- 某个职业 / 角色胜率、Ban/Pick 远高于平均
- 环境出现“唯一正确解”：只剩一套阵容 / 出装是可行解

## 产生原因（结构视角）

1. 数值成长和技能系数设计有明显超线
2. 机制价值远高于面板（控制、位移、安全性）
3. 新版本内容意外放大旧角色优势
4. 不同段位对角色强度感知差异

## 修复流程（可复用）

1. 数据侧先确认：胜率 + 使用率 + Ban 率分段表现
2. 优先处理破坏体验的极端问题（无解连招、无限控制等）
3. 再通过多次小幅数值调整微调强度
4. 按版本节奏分层：
   - 大版本可以动结构
   - 小版本尽量只调数值

> 平衡的目标不是“所有方案等强”，而是让玩家有多种“足够好”的选择。
