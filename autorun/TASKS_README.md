## 使用方式

### 方式1：使用 Orchestrator 自动处理

```bash
# 启动 Orchestrator
python autorun/orchestrator.py

# 在 Cursor 中激活 Worker
@cursor-worker 请处理 input 目录中的最新批次任务
```

### 方式2：手动处理单个任务

1. 查看 `tasks.json` 中的任务描述
2. 根据任务类型和描述执行相应操作
3. 完成后更新任务状态为 `done`

## 任务状态

- `pending`: 待处理
- `in_progress`: 处理中
- `done`: 已完成
- `failed`: 失败

## 注意事项

1. **内容补充任务**：需要访问GDC Vault、YouTube、Bilibili等平台查找资源
2. **新建文档任务**：需要搜索GDC官网、会议日程等找到具体演讲信息
3. **批量任务**：某些任务包含多个文档，可以分批处理
4. **质量要求**：所有文档必须符合 `docs/collection-plan-2025.md` 中的收集要求标准
